Pi-Hole adblocker with Raspberry Pi

I configured Raspberry Pi to work as DNS server of my home network blocking advertisement.

I really hate ads. I have been using adblocker extension in browser for long time, but some time ago I discovered quite popular project called Pi-hole. Pi-hole can be installed to Raspberry Pi and it acts as a DNS server for local network. Pi-hole then blocks DNS requests for known advertisement domains.

This is brilliant, because you are no longer relying on browser extension, but your whole network is blocked from ads. This includes smart-tv and mobile.

Pi-hole is really easy to setup and if you want to do it yourself, there are instructions and more information in Pi-hole's website and github page.
